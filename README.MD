# demo-server
### Описание
Простейший CRUD сервис на популярном стеке технологий: 
* Spring Boot
* Hibernate
* HikariCP
* Liquibase
* Swagger

### Запуск
  
 java -jar demo-0.0.1-SNAPSHOT.jar --spring.profiles.active={profile}
 
 Где profile = dev или prod
